# Nachbearbeitung: Modul 3

**Zeitaufwand:** 4-6 Stunden
**Deadline:** Vor Modul 4

## 🎯 Ziel

Vertiefen Sie die Konzepte aus Modul 3 durch praktische Hausaufgaben.

## 📝 Aufgaben

| Aufgabe | Thema | Dauer | Schwierigkeit |
|---------|-------|-------|---------------|
| 1 | Zahlenspiel | 2 Std | ⭐⭐ Mittel |
| 2 | Einkaufsliste | 1.5 Std | ⭐⭐ Mittel |
| 3 | Primzahlen-Finder | 2 Std | ⭐⭐⭐ Schwer |
| 4 | Reflexion | 30 Min | ⭐ Einfach |

## 📚 Aufgabenübersicht

### [Aufgabe 1: Zahlenspiel](./aufgabe-1-zahlenspiel.md) (2 Std)

Erstellen Sie ein interaktives Zahlenspiel:
- Zufallszahl generieren
- Benutzer raten lassen
- Hinweise geben (zu hoch/zu niedrig)
- Versuche zählen

### [Aufgabe 2: Einkaufsliste](./aufgabe-2-einkaufsliste.md) (1.5 Std)

Erstellen Sie einen Einkaufslisten-Manager:
- Artikel hinzufügen
- Artikel anzeigen
- Artikel löschen
- Menü-System

### [Aufgabe 3: Primzahlen-Finder](./aufgabe-3-primzahlen.md) (2 Std)

Erstellen Sie ein Programm, das Primzahlen findet:
- Primzahl-Test
- Alle Primzahlen bis N
- Statistiken

### [Aufgabe 4: Reflexion](./reflexion.md) (30 Min)

Reflektieren Sie Ihren Lernfortschritt.

## ✅ Abgabe

**Mindestanforderung:** 2 von 3 Aufgaben + Reflexion

**Empfohlen:** Alle 3 Aufgaben für maximalen Lerneffekt

## 💡 Tipps

- Beginnen Sie mit Aufgabe 1 (am einfachsten)
- Nutzen Sie die Beispiele aus [05-beispiele](../05-beispiele/)
- Testen Sie Ihr Programm ausführlich
- Experimentieren Sie mit Erweiterungen

- Materialien in [04-materialien](../04-materialien/)
- Python-Dokumentation
