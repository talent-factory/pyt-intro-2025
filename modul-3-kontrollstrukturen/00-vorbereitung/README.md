# Vorbereitung: Modul 3

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziel der Vorbereitung

Sie bereiten sich auf den dritten Kurstag vor, indem Sie:

1. Grundlagen von Kontrollstrukturen verstehen
2. If/elif/else Bedingungen kennenlernen
3. While- und For-Schleifen verstehen
4. Erste Experimente mit Schleifen durchführen
5. Ihr Wissen mit einem Quiz überprüfen

## 📚 Aufgaben

### 1. Leseauftrag (60-90 Min)

Lesen Sie die folgenden Dokumente:

- [leseauftrag.md](./leseauftrag.md) - Kontrollstrukturen, Bedingungen, Schleifen

**Themen:**
- Bedingte Anweisungen (if, elif, else)
- While-Schleifen für Wiederholungen
- For-Schleifen für Iteration
- Break und continue
- Listen-Grundlagen

### 2. Video-Tutorials (30-45 Min)

Schauen Sie sich folgende Videos an (optional, aber empfohlen):

- **[Python If-Else Statements](https://www.youtube.com/watch?v=f4KOjWS_KZs)** (12 Min)
  - Bedingte Anweisungen
  - If/elif/else Syntax

- **[Python While Loops](https://www.youtube.com/watch?v=ECduJk00mUU)** (10 Min)
  - While-Schleifen verstehen
  - Break und continue

- **[Python For Loops](https://www.youtube.com/watch?v=OnDr4J2UXSA)** (15 Min)
  - For-Schleifen und range()
  - Iteration über Listen

### 3. Erste Experimente (45-60 Min)

Arbeiten Sie durch:

- [erste-experimente.md](./erste-experimente.md) - Praktische Übungen mit Kontrollstrukturen

**Experimente:**
- Einfache if/else Bedingungen
- While-Schleifen mit Zählern
- For-Schleifen mit range()
- Listen durchlaufen

### 4. Quiz zur Selbstkontrolle (15-30 Min)

Testen Sie Ihr Wissen:

- [quiz.md](./quiz.md) - 10 Fragen zu Modul 3

## ✅ Checkliste

Vor dem Präsenztag sollten Sie:

- [ ] Leseauftrag vollständig gelesen
- [ ] Videos angeschaut (optional)
- [ ] Erste Experimente durchgeführt
- [ ] Quiz mit mindestens 7/10 Punkten bestanden

## 💡 Tipps

- **Wiederholen Sie Modul 2:** Boolsche Werte und Vergleiche sind wichtig
- **Experimentieren Sie:** Probieren Sie verschiedene Bedingungen aus
- **Klein anfangen:** Erst einfache, dann komplexe Schleifen
- **Fehler sind OK:** Endlosschleifen passieren - Ctrl+C beendet sie
- **Notieren Sie Fragen:** Schreiben Sie auf, was unklar ist
