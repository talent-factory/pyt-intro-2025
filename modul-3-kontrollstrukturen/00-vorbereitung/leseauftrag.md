# Leseauftrag: Kontrollstrukturen

**Zeitaufwand:** 60-90 Minuten

## 1. Bedingte Anweisungen (if/elif/else)

Programme treffen Entscheidungen mit if-Anweisungen.

```python
alter = 18
if alter >= 18:
    print("Volljährig")
```

## 2. While-Schleifen

Wiederholen Code solange Bedingung erfüllt ist.

```python
zaehler = 1
while zaehler <= 5:
    print(zaehler)
    zaehler += 1
```

## 3. For-Schleifen

Durchlaufen Sequenzen.

```python
for i in range(5):
    print(i)
```

## 4. Listen

Speichern mehrere Werte.

```python
namen = ["Anna", "Max", "Lisa"]
for name in namen:
    print(name)
```
