# Beispiele: Modul 3

Vollständige Beispielprogramme für Modul 3.

## 📚 Übersicht

| Datei | Beschreibung | Konzepte |
|-------|--------------|----------|
| `bedingungen.py` | If/elif/else Beispiele | Bedingte Anweisungen |
| `while_schleifen.py` | While-Schleifen | Wiederholungen mit Bedingung |
| `for_schleifen.py` | For-Schleifen | Iteration mit range() |
| `listen.py` | Listen-Operationen | Listen erstellen, ändern |
| `verschachtelt.py` | Verschachtelte Strukturen | Komplexe Logik |
| `zahlenraten.py` | Zahlenrate-Spiel | While, if, input |
| `einkaufsliste.py` | Einkaufslisten-Manager | Listen, Menü, Schleifen |
| `primzahlen.py` | Primzahlen-Finder | For, if, Mathematik |
| `muster.py` | Muster zeichnen | Verschachtelte Schleifen |
| `menu.py` | Menü-System | While, if-elif-else |
| `notensystem.py` | Punkte in Noten umwandeln | While, if-elif-else, Validierung |
| `gerade_zahlen.py` | Gerade Zahlen filtern | For, if, Listen, Modulo |
| `notenliste.py` | Notenstatistik berechnen | For, if, Listen, Statistik |
| `notensystem_improved.py` | Notensystem mit Datenstruktur | Listen, Tupel, For-Schleife |

## 🚀 Verwendung

Alle Programme können direkt ausgeführt werden:

```bash
python bedingungen.py
python while_schleifen.py
# etc.
```
