# Vorbereitung: Modul 1

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziele der Vorbereitung

Nach der Vorbereitung:

- ✅ Haben Sie eine funktionierende Entwicklungsumgebung (lokal oder Codespaces)
- ✅ Verstehen Sie grundlegend, was Programmieren bedeutet
- ✅ Haben Sie erste Erfahrungen mit Python gesammelt
- ✅ Sind Sie bereit für den Präsenztag

## 📋 Checkliste

### 1. Entwicklungsumgebung einrichten

**Wählen Sie eine Option:**

- [ ] **Option A:** [GitHub Codespaces einrichten](./codespaces-anleitung.md) (empfohlen für Anfänger)
- [ ] **Option B:** [Lokale Installation](./installationsanleitung.md) (für erfahrene Nutzer)

### 2. Leseauftrag

- [ ] [Was ist Programmieren?](./leseauftrag.md) lesen (30 Min.)
- [ ] Reflexionsfragen beantworten

### 3. Erste Schritte

- [ ] [Erste Schritte mit Python](./erste-schritte.md) durcharbeiten (60 Min.)

### 4. Quiz zur Selbstkontrolle

- [ ] [Quiz](./quiz.md) absolvieren (15 Min., min. 7/10 Punkte)

## ❓ Häufige Fragen

**Q: Muss ich alles verstehen?**  
A: Nein! Die Vorbereitung soll einen ersten Eindruck geben. Vieles wird im Präsenzunterricht vertieft.

**Q: Was, wenn die Installation nicht klappt?**  
A: Nutzen Sie GitHub Codespaces als Alternative. Wir helfen am Präsenztag.

**Q: Wie viel Vorwissen brauche ich?**  
A: Keines! Der Kurs ist für absolute Anfänger.

**Q: Kann ich die Videos überspringen?**  
A: Ja, sie sind optional. Aber sie helfen beim Verständnis.
