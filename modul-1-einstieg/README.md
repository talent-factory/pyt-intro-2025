# Modul 1: Einstieg in die Programmierung

**Dauer:** 1 Tag (4 Lektionen à 50 Minuten)
**Voraussetzung:** Keine

## 🎯 Lernziele

Nach diesem Modul können Sie:

1. ✅ Erklären, was Programmieren im Kern bedeutet und warum Python eine gute erste Sprache ist
2. ✅ Die Python Shell (REPL) und Jupyter Notebooks für einfache Experimente nutzen
3. ✅ VS Code als professionelle Entwicklungsumgebung einrichten und verwenden
4. ✅ Einfache Python-Programme schreiben und ausführen
5. ✅ Variablen deklarieren und sinnvoll benennen
6. ✅ Mit Zahlen (int, float) rechnen und Operatoren anwenden
7. ✅ Einfache Fehler erkennen und beheben
8. ✅ GitHub Codespaces als Alternative zur lokalen Installation nutzen

## 🎓 Wichtige Konzepte

In diesem Modul lernen Sie:

- **REPL** - Read-Eval-Print-Loop (interaktive Python Shell)
- **IDE** - Integrated Development Environment (VS Code)
- **Variablen** - Speicherplätze für Daten
- **Datentypen** - int (Ganzzahlen), float (Dezimalzahlen)
- **Operatoren** - +, -, *, /, //, %, **
- **Kommentare** - Dokumentation im Code
- **Fehler** - Syntax- und Laufzeitfehler

## 💡 Tipps für den Erfolg

- **Experimentieren Sie!** Die Python Shell ist perfekt zum Ausprobieren
- **Machen Sie Fehler!** Fehler sind der beste Lernweg
- **Fragen Sie!** Es gibt keine dummen Fragen
- **Üben Sie täglich!** Auch 15 Minuten helfen
- **Nutzen Sie Codespaces!** Wenn lokale Installation Probleme macht
