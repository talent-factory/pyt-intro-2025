# Praxis: Modul 1

**Dauer:** 4 Lektionen à 50 Minuten

## 📋 Übersicht

Jede Lektion folgt dem gleichen Aufbau:

- **Theorie** (15 Min.) - Konzepte verstehen
- **Live-Coding** (20 Min.) - Dozent zeigt Best Practices
- **Übung** (15 Min.) - Studierende üben selbst

## 📝 Für Dozenten

### Vorbereitung

- [ ] Beamer/Projektor testen
- [ ] Beispiel-Code vorbereiten
- [ ] Übungen ausdrucken (optional)
- [ ] Codespaces-Link bereit haben

### Während der Lektionen

- **Interaktiv:** Fragen stellen, Studierende einbeziehen
- **Live-Coding:** Fehler machen ist okay - zeigt den Lernprozess
- **Pausen:** Wichtig für Konzentration
- **Hilfe:** Bei Problemen individuell unterstützen

### Tipps

- Langsam sprechen und Code erklären
- Studierende zum Mittippen ermutigen
- Fehler als Lernchance nutzen
- Positive Atmosphäre schaffen
