# Vorbereitung: Modul 2

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziel der Vorbereitung

Sie bereiten sich auf den zweiten Kurstag vor, indem Sie:

1. Grundlagen zu Strings und Textverarbeitung lernen
2. Boolsche Werte und Vergleiche verstehen
3. Erste Experimente mit der Python Shell durchführen
4. Ihr Wissen mit einem Quiz überprüfen

### 1. Leseauftrag (60-90 Min)

- **[Boolsche Werte](https://studyflix.de/informatik/boolesche-operatoren-6897)** (10 Min)
  - True/False
  - Vergleichsoperatoren
  
- **[Input/Output](https://www.youtube.com/watch?v=W_Lf7m2r6XA)** (10 Min)
  - `input()` Funktion

### 3. Erste Experimente (45-60 Min)

Arbeiten Sie durch:

- [erste-experimente.md](./erste-experimente.md) - Praktische Übungen in der Python Shell

### 4. Quiz zur Selbstkontrolle (15-20 Min)

Testen Sie Ihr Wissen:

- [quiz.md](./quiz.md) - 10 Fragen zu Modul 2
