# Vorbereitung Modul 4

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziel der Vorbereitung

Sie bereiten sich auf den vierten Kurstag vor, indem Sie:

1. Grundlagen von Funktionen verstehen
2. Parameter und Rückgabewerte kennenlernen
3. Datenstrukturen (Listen, Dictionaries) verstehen
4. Erste Experimente mit Funktionen durchführen
5. Ihr Wissen mit einem Quiz überprüfen

## 📚 Aufgaben

### 1. Leseauftrag (60-90 Min)

Lesen Sie die folgenden Dokumente:

- [leseauftrag.md](./leseauftrag.md) - Funktionen & Datenstrukturen

**Themen:**
- Funktionen definieren mit `def`
- Parameter (positionell, benannt, default)
- Rückgabewerte mit `return`
- Listen vertiefen
- Dictionaries (Schlüssel-Wert-Paare)
- Tupel und Sets (kurze Einführung)

### 2. Video-Tutorials (30-45 Min)

Schauen Sie sich folgende Videos an (optional, aber empfohlen):

- **[Python Functions](https://www.youtube.com/watch?v=9Os0o3wzS_I)** (15 Min)
  - Funktionen definieren
  - Parameter und Return

- **[Python Dictionaries](https://www.youtube.com/watch?v=daefaLgNkw0)** (12 Min)
  - Dictionary-Grundlagen
  - Schlüssel-Wert-Paare

- **[Python Data Structures](https://www.youtube.com/watch?v=R-HLU9Fl5ug)** (15 Min)
  - Listen, Tupel, Sets im Überblick
  - Wann welche Struktur?

### 3. Erste Experimente (45-60 Min)

Arbeiten Sie durch:

- [erste-experimente.md](./erste-experimente.md) - Praktische Übungen mit Funktionen

**Experimente:**
- Einfache Funktionen schreiben
- Funktionen mit Parametern
- Rückgabewerte nutzen
- Dictionaries erstellen und verwenden

### 4. Quiz zur Selbstkontrolle (15-20 Min)

Testen Sie Ihr Wissen:

- [quiz.md](./quiz.md) - 10 Fragen zu Modul 4

## ✅ Checkliste

Vor dem Präsenztag sollten Sie:

- [ ] Leseauftrag vollständig gelesen
- [ ] Videos angeschaut (optional)
- [ ] Erste Experimente durchgeführt
- [ ] Quiz mit mindestens 7/10 Punkten bestanden
- [ ] Python-Umgebung funktioniert (VS Code oder Codespaces)
- [ ] Module 1-3 Konzepte wiederholt
- [ ] Fragen notiert für den Präsenztag

## 💡 Tipps

- **Wiederholen Sie Modul 3:** Listen sind wichtig für Modul 4
- **Funktionen sind mächtig:** Sie machen Code wiederverwendbar
- **Experimentieren Sie:** Probieren Sie verschiedene Parameter aus
- **Dokumentieren Sie:** Nutzen Sie Docstrings in Funktionen
- **Klein anfangen:** Erst einfache, dann komplexe Funktionen

## 🆘 Hilfe

Falls Sie Probleme haben:

1. Überprüfen Sie die [häufigen Fehler](../04-materialien/haeufige-fehler.md)
2. Wiederholen Sie Module 1-3
3. Nutzen Sie die Python-Dokumentation
4. Fragen Sie im Kurs nach
