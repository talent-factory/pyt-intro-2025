## Fragen

### 1. Was ist eine Funktion?

- [ ] A) Eine Variable
- [ ] B) Ein wiederverwendbarer Code-Block
- [ ] C) Eine Schleife
- [ ] D) Ein Datentyp

### 2. Wie definiert man eine Funktion in Python?

- [ ] A) `function name():`
- [ ] B) `def name():`
- [ ] C) `func name():`
- [ ] D) `define name():`

### 3. Was macht `return` in einer Funktion?

- [ ] A) Beendet das Programm
- [ ] B) Gibt einen Wert zurück
- [ ] C) Startet eine Schleife
- [ ] D) Druckt etwas aus

### 4. Welche Datenstruktur ist veränderbar?

- [ ] A) Tupel
- [ ] B) String
- [ ] C) Liste
- [ ] D) Integer

### 5. Wie greift man auf das erste Element einer Liste zu?

- [ ] A) `liste[1]`
- [ ] B) `liste[0]`
- [ ] C) `liste.first()`
- [ ] D) `liste{0}`

### 6. Was ist ein Dictionary?

- [ ] A) Eine geordnete Liste
- [ ] B) Eine Sammlung von Schlüssel-Wert-Paaren
- [ ] C) Eine Zahl
- [ ] D) Eine Funktion

### 7. Wie fügt man ein Element zu einer Liste hinzu?

- [ ] A) `liste.add(element)`
- [ ] B) `liste.append(element)`
- [ ] C) `liste.insert(element)`
- [ ] D) `liste + element`

### 8. Was ist der Unterschied zwischen Liste und Tupel?

- [ ] A) Kein Unterschied
- [ ] B) Tupel sind unveränderbar
- [ ] C) Listen sind schneller
- [ ] D) Tupel können nur Zahlen enthalten

### 9. Wie erstellt man ein leeres Dictionary?

- [ ] A) `dict = []`
- [ ] B) `dict = {}`
- [ ] C) `dict = ()`
- [ ] D) `dict = ""`

### 10. Was macht ein Set besonders?

- [ ] A) Es ist geordnet
- [ ] B) Es erlaubt Duplikate
- [ ] C) Es enthält nur eindeutige Elemente
- [ ] D) Es ist unveränderbar

## Lösungen

<details>
<summary>Klicken für Lösungen</summary>

1. **B** - Eine Funktion ist ein wiederverwendbarer Code-Block.
2. **B** - In Python definiert man Funktionen mit `def name():`.
3. **B** - `return` gibt einen Wert aus der Funktion zurück.
4. **C** - Listen sind veränderbar (mutable), Tupel und Strings nicht.
5. **B** - Indizierung beginnt in Python bei 0, also `liste[0]`.
6. **B** - Ein Dictionary speichert Schlüssel-Wert-Paare (`{"key": value}`).
7. **B** - `liste.append(element)` fügt am Ende ein Element hinzu.
8. **B** - Tupel sind unveränderbar, Listen veränderbar.
9. **B** - `{}` erstellt ein leeres Dictionary, `[]` wäre eine leere Liste.
10. **C** - Ein Set enthält nur eindeutige Elemente, Duplikate werden entfernt.

</details>
