# Hausaufgabe: Kontaktverwaltung


Erstellen Sie ein Programm zur Verwaltung von Kontakten.

**Anforderungen:**

1. Liste von Kontakten (Dictionaries mit name, email, telefon)
2. Funktionen:
   - `neuer_kontakt(name, email, telefon)` - Fügt Kontakt hinzu
   - `zeige_kontakte()` - Zeigt alle Kontakte
   - `suche_kontakt(name)` - Sucht Kontakt nach Name
   - `loesche_kontakt(name)` - Löscht Kontakt

**Beispiel-Ausgabe:**
```
KONTAKTVERWALTUNG
=================
1. Anna Muster (anna@example.com, 079 123 45 67)
2. Max Meier (max@example.com, 078 987 65 43)
```

**Zeitaufwand:** 2 Stunden

**Tipps:**
- Verwenden Sie eine globale Liste für Kontakte
- Nutzen Sie Dictionaries für jeden Kontakt
- Testen Sie alle Funktionen
