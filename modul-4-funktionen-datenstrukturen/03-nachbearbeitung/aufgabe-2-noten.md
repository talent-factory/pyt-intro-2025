# Hausaufgabe: Notenverwaltung


Erstellen Sie ein Notenverwaltungssystem.

**Anforderungen:**

1. Dictionary mit Studenten und ihren Noten
2. Funktionen:
   - `note_hinzufuegen(student, note)` - Fügt Note hinzu
   - `durchschnitt(student)` - Berechnet Durchschnitt
   - `beste_note(student)` - Gibt beste Note zurück
   - `klassendurchschnitt()` - Durchschnitt aller Studenten

**Beispiel:**
```python
noten = {
    "Anna": [5.5, 6.0, 5.0],
    "Max": [4.5, 5.0, 5.5]
}
```

**Zeitaufwand:** 2 Stunden

**Erweiterung:**
- Notenstatistik (Min, Max, Durchschnitt)
- Bestanden/Nicht bestanden (>= 4.0)
