# Nachbearbeitung: Modul 4

**Zeitaufwand:** 4-6 Stunden
**Deadline:** Vor Modul 5

## 🎯 Ziel

Vertiefen Sie die Konzepte aus Modul 4 durch praktische Hausaufgaben mit Funktionen und Datenstrukturen.

## 📝 Aufgaben

| Aufgabe | Thema | Dauer | Schwierigkeit |
|---------|-------|-------|---------------|
| 1 | Kontaktverwaltung | 2 Std | ⭐⭐ Mittel |
| 2 | Notenverwaltung | 2 Std | ⭐⭐ Mittel |
| 3 | Wörterbuch | 2 Std | ⭐⭐⭐ Schwer |
| 4 | Reflexion | 15 Min | ⭐ Einfach |

## 📚 Aufgabenübersicht

### [Aufgabe 1: Kontaktverwaltung](./aufgabe-1-kontakte.md) (2 Std)

Erstellen Sie eine Kontaktverwaltung mit Funktionen und Dictionaries:
- Kontakte hinzufügen (Name, Telefon, E-Mail)
- Kontakte anzeigen
- Kontakte suchen
- Kontakte löschen
- Menü-System

**Technologien:** Funktionen, Dictionaries, Listen

### [Aufgabe 2: Notenverwaltung](./aufgabe-2-noten.md) (2 Std)

Erstellen Sie ein Notenverwaltungssystem:
- Noten hinzufügen
- Durchschnitt berechnen
- Beste/schlechteste Note finden
- Notenverteilung anzeigen
- Statistiken

**Technologien:** Funktionen, Listen, Dictionaries

### [Aufgabe 3: Wörterbuch](./aufgabe-3-woerterbuch.md) (2 Std)

Erstellen Sie ein einfaches Wörterbuch-Programm:
- Wörter und Übersetzungen speichern
- Wörter nachschlagen
- Neue Wörter hinzufügen
- Wörter löschen
- Alle Wörter anzeigen

**Technologien:** Funktionen, Dictionaries, Sets

### [Aufgabe 4: Reflexion](./reflexion.md) (15 Min)

Reflektieren Sie Ihren Lernfortschritt.

## ✅ Abgabe

**Mindestanforderung:** 2 von 3 Aufgaben + Reflexion

**Empfohlen:** Alle 3 Aufgaben für maximalen Lerneffekt

### Format

- **Code:** Python-Dateien (`.py`)
- **Reflexion:** Markdown (`.md`) oder PDF
- **Ordner:** `modul-4-nachbearbeitung/`

### Struktur

```
modul-4-nachbearbeitung/
├── aufgabe-1-kontakte.py
├── aufgabe-2-noten.py
├── aufgabe-3-woerterbuch.py
└── reflexion.md
```

## 💡 Tipps

- **Funktionen nutzen:** Jede Aktion sollte eine eigene Funktion sein
- **Docstrings:** Dokumentieren Sie Ihre Funktionen
- **Testen Sie:** Probieren Sie verschiedene Szenarien
- **Modular denken:** Kleine Funktionen sind besser als grosse
