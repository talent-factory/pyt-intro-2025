# Hausaufgabe: Deutsch-Englisch Wörterbuch


Erstellen Sie ein interaktives Wörterbuch.

**Anforderungen:**

1. Dictionary mit Deutsch-Englisch Übersetzungen
2. Funktionen:
   - `uebersetze(wort)` - Übersetzt Deutsch → Englisch
   - `neues_wort(deutsch, englisch)` - Fügt Wort hinzu
   - `zeige_alle()` - Zeigt alle Wörter
   - `quiz()` - Zufälliges Wort abfragen

**Beispiel:**
```
Wörterbuch Deutsch-Englisch
============================
1. Übersetzen
2. Neues Wort
3. Alle Wörter
4. Quiz
5. Beenden
```

**Zeitaufwand:** 2 Stunden

**Erweiterung:**
- Speichern in Datei (später in Modul 5)
- Rückwärts-Übersetzung (Englisch → Deutsch)
