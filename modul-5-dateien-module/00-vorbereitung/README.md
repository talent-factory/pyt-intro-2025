# Vorbereitung: Modul 5

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziel der Vorbereitung

Sie bereiten sich auf den fünften und letzten Kurstag vor, indem Sie:

1. Grundlagen der Dateiverarbeitung verstehen
2. CSV- und JSON-Formate kennenlernen

4. Erste Experimente mit Dateien durchführen
5. Ihr Wissen mit einem Quiz überprüfen

**Experimente:**
- Textdateien erstellen und lesen

## 💡 Tipps

- **Sicherheit:** Arbeiten Sie mit Kopien, nicht mit wichtigen Originaldateien
- **Klein anfangen:** Testen Sie mit kleinen Beispieldateien
- **Context Manager:** Gewöhnen Sie sich an `with open()` statt `open()`/`close()`
- **Fehler erwarten:** Fehlerbehandlung ist normal und wichtig
- **Wiederholen Sie Module 1-4:** Falls nötig, frischen Sie Grundlagen auf

3. Nutzen Sie die Python-Dokumentation
