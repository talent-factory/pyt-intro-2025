## Fragen

### 1. Welcher Modus überschreibt eine Datei?

- [ ] A) `"r"`
- [ ] B) `"w"`
- [ ] C) `"a"`
- [ ] D) `"r+"`

### 2. Was macht `with open(...)`?

- [ ] A) Öffnet Datei permanent
- [ ] B) Schliesst Datei automatisch
- [ ] C) Löscht Datei
- [ ] D) Erstellt Datei

### 3. Wofür steht CSV?

- [ ] A) Computer System Values
- [ ] B) Comma-Separated Values
- [ ] C) Code Storage Values
- [ ] D) Central System Variables

### 4. Welches Format eignet sich am besten für verschachtelte, strukturierte Daten?

- [ ] A) TXT
- [ ] B) CSV
- [ ] C) JSON
- [ ] D) XML

### 5. Wie importiert man ein Modul?

- [ ] A) `load math`
- [ ] B) `import math`
- [ ] C) `include math`
- [ ] D) `use math`

### 6. Was macht `json.dump()`?

- [ ] A) Liest JSON
- [ ] B) Schreibt JSON in eine Datei
- [ ] C) Löscht JSON
- [ ] D) Prüft JSON

### 7. Wie liest man alle Zeilen einer Datei in eine Liste?

- [ ] A) `f.read()`
- [ ] B) `f.readlines()`
- [ ] C) `f.getlines()`
- [ ] D) `f.all()`

### 8. Wie erstellt man ein eigenes Modul?

- [ ] A) Eine `.py`-Datei erstellen
- [ ] B) Eine `.mod`-Datei erstellen
- [ ] C) Eine `.module`-Datei erstellen
- [ ] D) Eine `.lib`-Datei erstellen

### 9. Was macht `newline=""` bei CSV?

- [ ] A) Fügt Leerzeilen ein
- [ ] B) Verhindert zusätzliche Leerzeilen unter Windows
- [ ] C) Löscht Zeilen
- [ ] D) Hat keinen Effekt

### 10. Welches Modul nutzt man für Datum und Uhrzeit?

- [ ] A) `time`
- [ ] B) `date`
- [ ] C) `datetime`
- [ ] D) `calendar`

## Lösungen

<details>
<summary>Klicken für Lösungen</summary>

1. **B** - `"w"` (write) überschreibt die Datei. `"a"` würde anhängen.
2. **B** - `with open(...)` schliesst die Datei automatisch nach Gebrauch.
3. **B** - CSV = Comma-Separated Values (komma-getrennte Werte).
4. **C** - JSON eignet sich besonders gut für verschachtelte, strukturierte Daten.
5. **B** - `import math` importiert das Modul `math`.
6. **B** - `json.dump()` schreibt JSON-Daten in eine Datei (`json.dumps()` gibt einen String zurück).
7. **B** - `f.readlines()` liefert eine Liste mit allen Zeilen.
8. **A** - Eine `.py`-Datei kann direkt als Modul importiert werden.
9. **B** - `newline=""` verhindert zusätzliche Leerzeilen beim CSV-Schreiben unter Windows.
10. **C** - `datetime` ist das Standard-Modul für Datum und Uhrzeit.

</details>
