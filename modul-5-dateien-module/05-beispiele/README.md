# Beispiele: Modul 5

Vollständige Beispielprogramme für Modul 5.

## 📁 Programme

### Vollständige Beispiele

1. **[textdateien.py](./textdateien.py)**
   - Tagebuch-Eintrag hinzufügen
   - Einträge anzeigen
   - Datei analysieren

2. **[csv_beispiel.py](./csv_beispiel.py)**
   - Kontakte in CSV speichern
   - Kontakte anzeigen
   - Kontakt suchen

3. **[json_beispiel.py](./json_beispiel.py)**
   - Konfiguration erstellen
   - Einstellungen ändern
   - Verschachtelte Daten

### Weitere Programme

4. **[aufgabenliste.py](./aufgabenliste.py)** - Aufgabenverwaltung mit JSON
5. **[notenverwaltung.py](./notenverwaltung.py)** - Notenverwaltung mit CSV
6. **[datei_utils.py](./datei_utils.py)** - Datei-Utilities Modul
7. **[text_tools.py](./text_tools.py)** - Text-Verarbeitungs-Modul
8. **[backup_tool.py](./backup_tool.py)** - Backup-Tool für Dateien
9. **[log_analyzer.py](./log_analyzer.py)** - Log-Datei Analyzer
10. **[daten_konverter.py](./daten_konverter.py)** - CSV ↔ JSON Konverter

## 🚀 Verwendung

Alle Programme können direkt ausgeführt werden:

```bash
python textdateien.py
python csv_beispiel.py
python json_beispiel.py
```

- Dateien lesen und schreiben
- CSV-Dateien verarbeiten
- JSON-Daten speichern
- Eigene Module erstellen
