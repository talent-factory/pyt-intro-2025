# Aufgabe 3: Tool-Vergleich

**Zeitaufwand:** 2 Stunden
**Schwierigkeit:** ⭐⭐⭐ Schwer

## 🎯 Ziel

Probieren Sie eine Alternative zu Claude Code aus und vergleichen Sie beide Tools anhand derselben Programmieraufgabe.

## 📝 Anforderungen

### Basis-Features (Pflicht)

1. **Ein alternatives Tool wählen:**
   - GitHub Copilot (IDE-Integration)
   - Gemini CLI (Terminal-basiert)
   - ChatGPT (Weboberfläche)
   - Claude.ai (Weboberfläche)
2. **Dieselbe Aufgabe mit beiden Tools lösen**
   - Einfache Aufgabe wählen (z.B. Taschenrechner, Todo-Liste)
   - Zuerst mit Claude Code lösen
   - Dann mit dem alternativen Tool lösen
3. **Vergleichstabelle ausfüllen** (siehe unten)
4. **Persönliche Empfehlung** formulieren

### Erweiterte Features (Optional)

5. **Drittes Tool** testen und in den Vergleich aufnehmen
6. **Verschiedene Aufgabentypen** testen (Erklären, Debuggen, Generieren)

## 💡 Beispiel-Aufgabe für den Vergleich

Verwenden Sie z.B. diese Aufgabe für beide Tools:

```
"Erstelle ein Python-Programm, das eine einfache Todo-Liste
verwaltet. Der Benutzer soll Aufgaben hinzufügen, als erledigt
markieren und alle Aufgaben anzeigen können."
```

## 🔧 Tool-Optionen

| Tool | Kategorie | Zugang |
|------|-----------|--------|

- Verschiedene KI-Tools kennenlernen

- Das passende Tool für die Aufgabe wählen
- Bewertungskriterien anwenden

## 💻 Hilfe

### Fairer Vergleich

- Verwenden Sie den **exakt gleichen Prompt** für beide Tools
- Geben Sie beiden Tools **gleich viel Zeit**
- Bewerten Sie die **erste Antwort** (nicht nach 10 Nachfragen)
- Testen Sie den Code beider Tools **tatsächlich**

### Bewertungskriterien erklärt

| Kriterium | 1 Stern | 5 Sterne |
|-----------|---------|----------|
| Benutzerfreundlichkeit | Kompliziert, unverständlich | Intuitiv, einfach |
| Code-Qualität | Fehlerhaft, unlesbar | Sauber, gut kommentiert |
| Geschwindigkeit | Sehr langsam | Sofortige Antwort |
| Erklärungen | Keine oder verwirrend | Klar und hilfreich |
| Fehlerbehandlung | Ignoriert Fehler | Erkennt und behebt Fehler |
