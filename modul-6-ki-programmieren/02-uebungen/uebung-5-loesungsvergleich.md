# Übung 5: Lösungsvergleich

**Dauer:** 15 Minuten
**Schwierigkeit:** ⭐⭐ Mittel

## 📝 Aufgabe

Wählen Sie eine Aufgabe aus Modul 3 oder 4 und erstellen Sie **zwei Lösungen**: eine selbst geschriebene und eine von Claude Code generierte. Vergleichen Sie anschliessend beide Lösungen.

### Schritt 1: Aufgabe auswählen

Wählen Sie eine Aufgabe, die Sie bereits gelöst haben, z.B.:

- Einkaufslisten-Verwaltung (Modul 4)
- Zahlenraten-Spiel (Modul 3)
- Kontaktbuch (Modul 4)

### Schritt 2: Eigene Lösung bereithalten

Nehmen Sie Ihre bestehende Lösung oder schreiben Sie eine neue.

### Schritt 3: KI-Lösung generieren

Geben Sie Claude Code die gleiche Aufgabenstellung und lassen Sie eine Lösung generieren.

### Schritt 4: Vergleichstabelle ausfüllen

| Kriterium | Eigene Lösung | KI-Lösung | Besser? |
|-----------|--------------|-----------|---------|
| **Lesbarkeit** | z.B. einfache Struktur | z.B. gut kommentiert | Eigene / KI |
| **Korrektheit** | z.B. alle Fälle abgedeckt | z.B. Randfälle fehlen | Eigene / KI |
| **Effizienz** | z.B. einfache Schleife | z.B. eleganter Ansatz | Eigene / KI |
| **Verständlichkeit** | z.B. sofort verständlich | z.B. neue Konzepte | Eigene / KI |
| **Fehlerbehandlung** | z.B. keine | z.B. try/except | Eigene / KI |

## 💡 Reflexionsfragen

- Welche Lösung würden Sie in einem echten Projekt verwenden?
- Was haben Sie von der KI-Lösung gelernt?
- Gibt es Teile, die Sie in Ihre Lösung übernehmen würden?
- War Ihre eigene Lösung in bestimmten Bereichen besser? Warum?
