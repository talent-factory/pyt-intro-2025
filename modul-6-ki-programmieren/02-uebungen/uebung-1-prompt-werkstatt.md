# Übung 1: Prompt-Werkstatt

**Dauer:** 15 Minuten
**Schwierigkeit:** ⭐ Einfach

## 📝 Aufgabe

Nehmen Sie 3 Aufgaben aus Modul 3 (z.B. Notensystem, Zahlenraten, Primzahlen) und formulieren Sie für jede Aufgabe **einen guten** und **einen schlechten** Prompt.

### Schritt 1: Schlechten Prompt formulieren

Schreiben Sie einen vagen, unspezifischen Prompt:

```
Schlechter Prompt:
"Mach ein Notenprogramm"
```

### Schritt 2: Guten Prompt formulieren

Schreiben Sie einen spezifischen, kontextreichen Prompt:

```
Guter Prompt:
"Erstelle ein Python-Programm notensystem.py, das eine Punktzahl (0-100)
als Eingabe nimmt und die Schweizer Note (1-6) ausgibt.
Notenskala:
- 90-100: Note 6
- 80-89: Note 5
- 70-79: Note 4
- 60-69: Note 3
- 0-59: Note 2
Verwende if/elif/else. Fange ungültige Eingaben ab.
Kommentare und Variablennamen auf Deutsch."
```

### Schritt 3: Beide Prompts testen

Geben Sie beide Prompts an Claude Code und vergleichen Sie die Ergebnisse.

## 💡 Aufgabenvorschläge

| # | Aufgabe | Modul |
|---|---------|-------|
| 1 | Notensystem | Modul 3 |
| 2 | Zahlenraten | Modul 3 |
| 3 | Primzahlen prüfen | Modul 3 |

## 📋 Bewertungs-Checkliste für gute Prompts

Prüfen Sie jeden Prompt mit dieser Checkliste:

- [ ] **Spezifisch:** Ist klar, was genau erstellt werden soll?
- [ ] **Kontextreich:** Sind alle nötigen Details enthalten (Sprache, Konventionen)?
- [ ] **Klares Ziel:** Ist das erwartete Ergebnis beschrieben?
- [ ] **Beispiele:** Sind Ein-/Ausgabe-Beispiele enthalten?
- [ ] **Einschränkungen:** Sind Randbedingungen definiert (z.B. Fehlerbehandlung)?

## ✅ Checkliste

- [ ] 3 Aufgaben ausgewählt
- [ ] Je einen schlechten Prompt formuliert
- [ ] Je einen guten Prompt formuliert
- [ ] Prompts getestet und Ergebnisse verglichen
- [ ] Unterschiede in den Ergebnissen dokumentiert
