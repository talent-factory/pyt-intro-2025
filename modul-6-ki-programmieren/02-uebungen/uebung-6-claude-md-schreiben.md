# Übung 6: CLAUDE.md schreiben

**Dauer:** 15 Minuten
**Schwierigkeit:** ⭐⭐⭐ Schwer

## 📝 Aufgabe

Erstellen Sie eine `CLAUDE.md`-Datei für ein fiktives Mini-Projekt: **"Notenverwaltung für eine Schule"**. Diese Datei soll einem KI-Assistenten alle nötigen Informationen geben, um im Projekt produktiv mitarbeiten zu können.

### Projektbeschreibung

Das fiktive Projekt "Notenverwaltung" hat folgende Eckdaten:

- **Zweck:** Noten von Schülerinnen und Schülern verwalten
- **Funktionen:** Noten erfassen, Durchschnitt berechnen, Zeugnisse erstellen
- **Technologie:** Python 3.11, SQLite-Datenbank
- **Benutzer:** Lehrpersonen an einer Schweizer Sekundarschule

### Schritt 1: Überlegen Sie, welche Informationen die KI braucht

- Projektbeschreibung und Zweck
- Ordnerstruktur
- Namenskonventionen (Variablen, Dateien)
- Code-Stil und Sprache
- Wichtige Regeln und Einschränkungen
- Zielgruppe des Projekts

### Schritt 2: CLAUDE.md verfassen

Schreiben Sie die `CLAUDE.md`-Datei mit mindestens diesen Abschnitten:

```markdown
# CLAUDE.md - Notenverwaltung

## Projektübersicht
[Was macht das Projekt? Für wen?]

## Projektstruktur
[Ordner und Dateien]

## Namenskonventionen
[Wie werden Variablen, Funktionen, Dateien benannt?]

## Code-Stil
[Sprache, Kommentare, Formatierung]

## Wichtige Regeln
[Was muss beachtet werden?]
```

### Schritt 3: CLAUDE.md testen

Geben Sie Ihre `CLAUDE.md` an Claude Code und bitten Sie um eine einfache Funktion für das Projekt. Prüfen Sie, ob der generierte Code Ihren Konventionen entspricht.

## 📋 Checkliste für eine gute CLAUDE.md

- [ ] **Projektübersicht:** Zweck und Zielgruppe klar beschrieben?
- [ ] **Struktur:** Ordnerstruktur vollständig dokumentiert?
- [ ] **Konventionen:** Namensregeln für Variablen und Dateien definiert?
- [ ] **Code-Stil:** Sprache für Kommentare und Variablen festgelegt?
- [ ] **Regeln:** Wichtige Dos und Don'ts aufgelistet?
- [ ] **Beispiele:** Mindestens ein Code-Beispiel für den gewünschten Stil?

## ✅ Checkliste

- [ ] CLAUDE.md-Datei geschrieben
- [ ] Alle Abschnitte enthalten
- [ ] Mit Claude Code getestet
- [ ] Ergebnis bewertet: Hat die KI die Konventionen eingehalten?
