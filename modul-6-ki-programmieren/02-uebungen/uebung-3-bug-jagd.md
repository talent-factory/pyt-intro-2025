# Übung 3: Bug-Jagd mit KI

**Dauer:** 15 Minuten
**Schwierigkeit:** ⭐⭐ Mittel

## 📝 Aufgabe

In der Datei [`../05-beispiele/code_mit_bugs.py`](../05-beispiele/code_mit_bugs.py) finden Sie 3 Funktionen mit versteckten Bugs. Bitten Sie Claude Code um Hilfe beim Debuggen und dokumentieren Sie die Ergebnisse.

### Schritt 1: Code anschauen

Öffnen Sie die Datei `code_mit_bugs.py` und versuchen Sie, die Bugs **zuerst selbst** zu finden.

### Schritt 2: KI um Hilfe bitten

Geben Sie jede Funktion einzeln an Claude Code:

```
Finde die Bugs in dieser Funktion und erkläre, was falsch ist:

[Funktion hier einfügen]
```

### Schritt 3: Ergebnisse dokumentieren

Füllen Sie für jede Funktion diese Tabelle aus:

| Funktion | Bug selbst gefunden? | Bug von KI gefunden? | Erklärung der KI |
|----------|---------------------|---------------------|-------------------|
| Funktion 1 | Ja / Nein | Ja / Nein | ... |
| Funktion 2 | Ja / Nein | Ja / Nein | ... |
| Funktion 3 | Ja / Nein | Ja / Nein | ... |

## 💡 Tipps

- Versuchen Sie unterschiedliche Prompts, wenn die KI den Bug nicht sofort findet
- Fragen Sie nach: "Bist du sicher, dass das alles ist?"
- Manchmal hilft es, den erwarteten Output mitzugeben

## 📋 Reflexion

- Welchen Bug hat die KI sofort gefunden?
- Welchen Bug hat die KI übersehen?
- War Ihre eigene Bug-Suche oder die KI-Suche erfolgreicher?

## ✅ Checkliste

- [ ] Claude Code um Hilfe gebeten
- [ ] Ergebnisse in der Tabelle dokumentiert
