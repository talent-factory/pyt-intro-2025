# Beispiele Modul 6

Vollständige Beispielprogramme für Modul 6.

## 📚 Übersicht

| Datei | Beschreibung | Konzepte |
|-------|--------------|----------|
| `prompt_beispiele.py` | Gute vs. schlechte Prompts als Code-Kommentare | Prompting, Spezifität |
| `code_mit_bugs.py` | Funktionen mit versteckten Fehlern | Debugging, Fehlersuche |
| `ki_generiert_beispiel.py` | KI-generierte Lösung mit Annotationen | Code-Bewertung |
| `vorher_nachher.py` | Eigene vs. KI-Lösung im Vergleich | Vergleich, Bewertung |
| `claude_md_beispiel.py` | Einfluss von CLAUDE.md auf KI-Output | Projektkontext |
| `passwort_generator.py` | Musterlösung KI-gestützte Entwicklung | Code-Generierung |
| `refactoring_beispiel.py` | Code vor/nach KI-Refactoring | Refactoring |
| `ki_grenzen.py` | Wo KI versagt | Grenzen, Halluzination |
| `iteratives_prompting.py` | Schrittweiser Verfeinerungsprozess | Iteratives Prompting |
| `debugging_mit_ki.py` | Bugs finden mit KI-Unterstützung | Debugging |

## 🚀 Verwendung

Alle Programme können direkt ausgeführt werden:

```bash
python prompt_beispiele.py
python code_mit_bugs.py
# etc.
```

## 💡 Lernhinweise

- Lesen Sie den Code sorgfältig
- Führen Sie die Programme aus
- Experimentieren Sie mit Änderungen
- Nutzen Sie die Programme als Vorlage

## 📖 Kommentare

Alle Programme sind ausführlich kommentiert und erklären:
- Was der Code macht
- Warum bestimmte Entscheidungen getroffen wurden
- Welche Konzepte verwendet werden
