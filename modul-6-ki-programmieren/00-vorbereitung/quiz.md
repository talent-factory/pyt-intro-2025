# Quiz: Selbsttest Modul 6

**Bestehensgrenze:** 7 von 10 Punkten

## Frage 1

Was ist ein "Prompt"?

- [ ] A) Ein Programmierbefehl in Python
- [x] B) Eine Anweisung oder Frage an eine KI
- [ ] C) Ein Fehler im Code
- [ ] D) Ein Python-Modul

## Frage 2

- [ ] A) Chat-basiertes Tool
- [x] B) IDE-Integration
- [ ] C) CLI-Tool
- [ ] D) Betriebssystem

## Frage 3

Was bedeutet "Halluzination" bei KI?

- [ ] A) Die KI hat einen Virus
- [ ] B) Die KI arbeitet zu langsam
- [x] C) Die KI erfindet Informationen, die nicht stimmen
- [ ] D) Die KI kann keine Bilder anzeigen

## Frage 4

Wann sollte man KI-generierten Code prüfen?

- [ ] A) Nur bei komplexem Code
- [ ] B) Nur wenn er nicht funktioniert
- [ ] C) Nur bei sicherheitsrelevantem Code
- [x] D) Immer

## Frage 5

- [ ] A) Es ist immer kostenlos
- [ ] B) Es braucht kein Internet
- [x] C) Es kann direkt auf Projektdateien zugreifen
- [ ] D) Es ist schneller

## Frage 6

Was ist CLAUDE.md?

- [ ] A) Ein Python-Programm
- [ ] B) Eine Installationsdatei
- [x] C) Eine Datei, die dem KI-Tool Projektkontext gibt
- [ ] D) Eine Fehlermeldung

## Frage 7

Kann KI immer korrekten Code generieren?

- [ ] A) Ja, KI macht keine Fehler
- [x] B) Nein, KI-Code kann Fehler enthalten und muss geprüft werden
- [ ] C) Ja, wenn man den richtigen Prompt verwendet
- [ ] D) Nur bei einfachen Aufgaben

## Frage 8

Was bedeutet "iteratives Prompting"?

- [ ] A) Einen sehr langen Prompt schreiben
- [ ] B) Immer den gleichen Prompt verwenden
- [x] C) Den Prompt schrittweise verfeinern, bis das Ergebnis passt
- [ ] D) Mehrere KI-Tools gleichzeitig nutzen

## Frage 9

Welches Tool arbeitet direkt im Projektkontext (kann Dateien lesen und ändern)?

- [ ] A) ChatGPT im Browser
- [ ] B) Google Gemini
- [x] C) Claude Code (CLI)
- [ ] D) Wikipedia

## Frage 10

Was ist die wichtigste Regel beim KI-gestützten Programmieren?

- [ ] A) Immer den teuersten Plan kaufen
- [ ] B) KI-Code nie verändern
- [ ] C) Nur eine KI verwenden
- [x] D) Den generierten Code verstehen und prüfen

---

## Lösungen

1. B — Ein Prompt ist eine Anweisung oder Frage an eine KI
2. B — GitHub Copilot ist eine IDE-Integration
3. C — Halluzination bedeutet, dass die KI falsche Informationen erfindet
4. D — KI-generierter Code sollte immer geprüft werden
5. C — CLI-Tools können direkt auf Projektdateien zugreifen
6. C — CLAUDE.md gibt dem KI-Tool Kontext über das Projekt
7. B — KI-Code kann Fehler enthalten und muss immer geprüft werden
8. C — Iteratives Prompting bedeutet schrittweises Verfeinern des Prompts
9. C — Claude Code arbeitet direkt im Projektkontext
10. D — Den generierten Code verstehen und prüfen ist die wichtigste Regel
