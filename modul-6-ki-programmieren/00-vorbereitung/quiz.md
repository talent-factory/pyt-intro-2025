# Quiz: KI-gestütztes Programmieren

**Dauer:** 15 Minuten
**Bestanden:** 7/10 richtig

## 📝 Anleitung

- Beantworten Sie alle Fragen
- Notieren Sie Ihre Antworten
- Vergleichen Sie mit den Lösungen am Ende
- Wiederholen Sie bei weniger als 7 Punkten den Leseauftrag

## Fragen

### 1. Was ist ein "Prompt"?

- [ ] A) Ein Programmierbefehl in Python
- [ ] B) Eine Anweisung oder Frage an eine KI
- [ ] C) Ein Fehler im Code
- [ ] D) Ein Python-Modul

### 2. Welche Kategorie von KI-Tool ist GitHub Copilot?

- [ ] A) Chat-basiertes Tool
- [ ] B) IDE-Integration
- [ ] C) CLI-Tool
- [ ] D) Betriebssystem

### 3. Was bedeutet "Halluzination" bei KI?

- [ ] A) Die KI hat einen Virus
- [ ] B) Die KI arbeitet zu langsam
- [ ] C) Die KI erfindet Informationen, die nicht stimmen
- [ ] D) Die KI kann keine Bilder anzeigen

### 4. Wann sollte man KI-generierten Code prüfen?

- [ ] A) Nur bei komplexem Code
- [ ] B) Nur wenn er nicht funktioniert
- [ ] C) Nur bei sicherheitsrelevantem Code
- [ ] D) Immer

### 5. Was ist der Vorteil eines CLI-Tools (wie Claude Code) gegenüber einem Chat-Tool?

- [ ] A) Es ist immer kostenlos
- [ ] B) Es braucht kein Internet
- [ ] C) Es kann direkt auf Projektdateien zugreifen
- [ ] D) Es ist schneller

### 6. Was ist CLAUDE.md?

- [ ] A) Ein Python-Programm
- [ ] B) Eine Installationsdatei
- [ ] C) Eine Datei, die dem KI-Tool Projektkontext gibt
- [ ] D) Eine Fehlermeldung

### 7. Kann KI immer korrekten Code generieren?

- [ ] A) Ja, KI macht keine Fehler
- [ ] B) Nein, KI-Code kann Fehler enthalten und muss geprüft werden
- [ ] C) Ja, wenn man den richtigen Prompt verwendet
- [ ] D) Nur bei einfachen Aufgaben

### 8. Was bedeutet "iteratives Prompting"?

- [ ] A) Einen sehr langen Prompt schreiben
- [ ] B) Immer den gleichen Prompt verwenden
- [ ] C) Den Prompt schrittweise verfeinern, bis das Ergebnis passt
- [ ] D) Mehrere KI-Tools gleichzeitig nutzen

### 9. Welches Tool arbeitet direkt im Projektkontext (kann Dateien lesen und ändern)?

- [ ] A) ChatGPT im Browser
- [ ] B) Google Gemini
- [ ] C) Claude Code (CLI)
- [ ] D) Wikipedia

### 10. Was ist die wichtigste Regel beim KI-gestützten Programmieren?

- [ ] A) Immer den teuersten Plan kaufen
- [ ] B) KI-Code nie verändern
- [ ] C) Nur eine KI verwenden
- [ ] D) Den generierten Code verstehen und prüfen

## Lösungen

<details>
<summary>Klicken für Lösungen</summary>

1. **B** - Ein Prompt ist eine Anweisung oder Frage an eine KI.
2. **B** - GitHub Copilot ist eine IDE-Integration (Plugin für VS Code etc.).
3. **C** - Halluzination bedeutet, dass die KI falsche Informationen erfindet.
4. **D** - KI-generierter Code sollte immer geprüft werden.
5. **C** - CLI-Tools können direkt auf Projektdateien zugreifen und sie bearbeiten. Hinweis: Auch CLI-Tools benötigen weiterhin eine Internetverbindung zum KI-Server — der Vorteil liegt im Projektkontext, nicht in der Offline-Nutzung.
6. **C** - CLAUDE.md gibt dem KI-Tool Kontext über das Projekt.
7. **B** - KI-Code kann Fehler enthalten und muss immer geprüft werden.
8. **C** - Iteratives Prompting bedeutet schrittweises Verfeinern des Prompts.
9. **C** - Claude Code arbeitet direkt im Projektkontext.
10. **D** - Den generierten Code verstehen und prüfen ist die wichtigste Regel.

</details>
