# Vorbereitung Modul 6

**Zeitaufwand:** 2-3 Stunden
**Deadline:** Vor dem Präsenztag

## 🎯 Ziel der Vorbereitung

Sie bereiten sich auf den sechsten Kurstag vor, indem Sie:

1. Verstehen, was KI-gestütztes Programmieren bedeutet
2. Die drei Kategorien von KI-Tools kennenlernen (Chat, IDE, CLI)
3. Grundlagen des Promptings verstehen
4. Erste Experimente mit einem KI-Chat-Tool durchführen
5. Ihr Wissen mit einem Quiz überprüfen

## 📚 Aufgaben

### 1. Leseauftrag (60-90 Min)

Lesen Sie die folgenden Dokumente:

- [leseauftrag.md](./leseauftrag.md) - KI-gestütztes Programmieren verstehen

**Themen:**
- Was ist KI-gestütztes Programmieren?
- Drei Wege: Chat, IDE, CLI
- Was ist ein Prompt?
- Ethik und Verantwortung

### 2. Erste Experimente (45-60 Min)

Arbeiten Sie durch:

- [erste-experimente.md](./erste-experimente.md) - Praktische Experimente mit KI-Chat-Tools

**Experimente:**
- KI einen Python-Begriff erklären lassen
- KI Code schreiben lassen und ausprobieren
- Eigene Frage stellen und Antwort bewerten

### 3. Quiz zur Selbstkontrolle (15-30 Min)

Testen Sie Ihr Wissen:

- [quiz.md](./quiz.md) - 10 Fragen zu Modul 6

## ✅ Checkliste

Vor dem Präsenztag sollten Sie:

- [ ] Leseauftrag vollständig gelesen
- [ ] Erste Experimente durchgeführt
- [ ] Quiz mit mindestens 7/10 Punkten bestanden
- [ ] GitHub-Account vorhanden und funktionsfähig
- [ ] Claude Code Account vorbereitet (Anthropic-Account oder GitHub-Login — Details vom Dozenten)
- [ ] Module 1-5 Konzepte wiederholt
- [ ] Fragen notiert für den Präsenztag

## 💡 Tipps

- **Wiederholen Sie Module 1-5:** Sie werden mit KI an bekannten Konzepten arbeiten
- **Experimentieren Sie frei:** Probieren Sie verschiedene Fragen an die KI aus
- **Kritisch bleiben:** Hinterfragen Sie KI-Antworten immer
- **Keine Angst vor Fehlern:** Falsche Prompts sind Lernchancen
- **Notieren Sie Fragen:** Schreiben Sie auf, was unklar ist

## 🆘 Hilfe

Falls Sie Probleme haben:

1. Wiederholen Sie Module 1-5
2. Nutzen Sie die Python-Dokumentation
3. Probieren Sie verschiedene KI-Tools aus (Claude.ai, ChatGPT)
4. Fragen Sie im Kurs nach
