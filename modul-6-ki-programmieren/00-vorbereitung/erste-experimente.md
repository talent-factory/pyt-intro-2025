# Erste Experimente

**Zeitaufwand:** 45-60 Minuten

> **Hinweis:** Für diese Experimente verwenden Sie ein **kostenloses** KI-Chat-Tool wie [Claude.ai](https://claude.ai) oder [ChatGPT](https://chat.openai.com). Am Kurstag werden wir Claude Code (ein CLI-Tool) kennenlernen. Dafür benötigen Sie einen Account — Details erfahren Sie vom Dozenten.

## Experiment 1: Einen Python-Begriff erklären lassen

Geben Sie folgenden Prompt in ein KI-Chat-Tool ein:

> "Erkläre mir, was eine Variable in Python ist. Erkläre es so, als wäre ich ein absoluter Anfänger. Gib mir drei einfache Beispiele."

**Aufgaben:**
1. Lesen Sie die Antwort aufmerksam durch
2. Vergleichen Sie mit dem, was Sie in Modul 1 gelernt haben
3. Ist die Erklärung korrekt und verständlich?

**Reflexion:**
- Was hat die KI gut erklärt?
- Gab es etwas, das Sie anders erklären würden?
- Waren die Beispiele hilfreich?

## Experiment 2: Code generieren lassen

Geben Sie folgenden Prompt ein:

> "Schreibe mir eine Python-Funktion namens `ist_gerade`, die prüft ob eine Zahl gerade ist. Die Funktion soll True oder False zurückgeben. Verwende deutsche Kommentare."

**Aufgaben:**
1. Kopieren Sie den generierten Code
2. Erstellen Sie eine neue Python-Datei und fügen Sie den Code ein
3. Testen Sie die Funktion mit verschiedenen Zahlen:
   ```python
   print(ist_gerade(4))    # Erwartet: True
   print(ist_gerade(7))    # Erwartet: False
   print(ist_gerade(0))    # Erwartet: True
   ```
4. Funktioniert der Code korrekt?

**Reflexion:**
- Hat der Code auf Anhieb funktioniert?
- Sind die Kommentare verständlich?
- Hätten Sie den Code selbst schreiben können?

## Experiment 3: Eigene Frage stellen

Denken Sie an ein Thema aus Modul 1-5, das Sie noch nicht ganz verstanden haben, oder das Sie vertiefen möchten. Stellen Sie der KI eine Frage dazu.

**Beispiele für Fragen:**
- "Was ist der Unterschied zwischen einer Liste und einem Dictionary?"
- "Wie funktioniert eine verschachtelte Schleife?"
- "Erkläre mir, was `if __name__ == '__main__':` bedeutet"

**Aufgaben:**
1. Formulieren Sie Ihre eigene Frage
2. Geben Sie die Frage in das KI-Tool ein
3. Lesen Sie die Antwort und prüfen Sie sie
4. Dokumentieren Sie Frage und Antwort

**Reflexion:**
- War die Antwort hilfreich?
- War etwas in der Antwort unklar oder falsch?
- Wie könnten Sie den Prompt verbessern, um eine bessere Antwort zu erhalten?

## Zusammenfassung

Nach diesen drei Experimenten haben Sie:
- ✅ Ein KI-Chat-Tool ausprobiert
- ✅ Erfahren, wie KI Konzepte erklären kann
- ✅ KI-generierten Code getestet
- ✅ Eigene Fragen gestellt und Antworten bewertet

**Wichtige Erkenntnis:** KI kann hilfreich sein, aber Sie müssen die Antworten immer kritisch prüfen!
